# UI 优化修复清单

## 修复内容

### 1. 标签页文本截断问题 ✅
**问题**: "发布内容"、"Cookie 管理"、"发布历史" 显示不全

**修复**:
- 增加标签页最小宽度：`min-width: 120px`
- 增加内边距：`padding: 15px 30px`
- 设置窗口最小宽度：`setMinimumWidth(1300)`

### 2. 平台卡片颜色优化 ✅
**问题**: 选中状态背景色太鲜艳，文字看不清

**修复**:
- 知乎：从 `#0084ff` 改为 `#4a90e2`（更柔和）
- 小红书：从 `#ff2442` 改为 `#ff6b8a`（更柔和）
- 快手：保持 `#ff4906`
- 抖音：保持 `#00f0ff`
- 选中状态文字颜色改为白色，增加对比度

### 3. 发布按钮优化 ✅
**问题**: 渐变紫色不够醒目

**修复**:
- 改为更醒目的渐变色：`#667eea` → `#764ba2`
- 增加按钮高度：`50px`
- 增加字体大小：`16px`
- 添加图标：`🚀`

### 4. 整体布局优化 ✅
**问题**: 间距不协调

**修复**:
- 统一间距：`20px`
- 增加卡片圆角：`16px`
- 优化阴影效果
