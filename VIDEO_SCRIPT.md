### 第五章：Cookie 配置（5:30 - 6:30）

**【画面】**
- 切换到设置页面
- 展示 Cookie 配置说明
- 演示浏览器获取 Cookie 过程
- 粘贴 Cookie 到输入框

**【旁白】**
```
如果你想实现真正的自动发布，
需要配置各平台的 Cookie。

Cookie 是网站的登录凭证，
让程序能够模拟你的登录状态。

配置步骤很简单：

第一步，使用浏览器登录目标平台，
比如知乎。

第二步，按 F12 打开开发者工具。

第三步，找到 Application 标签，
展开 Cookies 选项。

第四步，复制 Cookie 值。

第五步，回到应用，进入设置页面，
将 Cookie 粘贴到对应平台的输入框中。

最后，点击保存配置按钮。

就完成了！

注意：Cookie 包含账号信息，
请妥善保管，不要分享给他人。
建议定期更新，确保安全。
```

**【Cookie 获取步骤提示框】**
```
获取 Cookie 五步法：
1. 登录平台
2. 按 F12 打开开发者工具
3. 找到 Application → Cookies
4. 复制 Cookie 值
5. 粘贴到应用设置中

⚠️ 安全提示：
- 不要分享 Cookie
- 定期更新
- 如泄露立即修改密码
```

## 🎨 制作建议

### 录制设置

**屏幕录制**:
- 分辨率：1920x1080
- 帧率：30fps 或 60fps
- 格式：MP4

**音频录制**:
- 麦克风：建议使用外置麦克风
- 环境：安静无回声
- 音量：适中，避免爆音

### 后期制作

**视频编辑**:
- 添加转场效果
- 插入文字说明
- 高亮鼠标点击
- 添加背景音乐（轻柔）

**字幕**:
- 添加中文字幕
- 关键步骤添加文字提示
- 重要信息添加标注框

### 视觉元素

**配色方案**:
- 主色：#667eea（紫色渐变）
- 辅色：#764ba2（紫色）
- 强调色：#10b981（绿色成功）
- 警告色：#ff6b6b（红色警告）

**字体**:
- 标题：Microsoft YaHei Bold
- 正文：Microsoft YaHei Regular
- 代码：Consolas
