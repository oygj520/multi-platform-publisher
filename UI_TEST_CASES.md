## UI 结构说明

当前 UI 包含两级标签页结构：

### 主标签页 (3 个)
- 📝 发布内容
- 📜 发布历史  
- ⚙️ 设置

### 设置页内的平台标签页 (4 个)
- 📝 知乎
- 📕 小红书
- 📹 快手
- 🎵 抖音

## 测试环境准备

### 前置条件
- [ ] Python 环境已安装 (Python 3.8+)
- [ ] PyQt5 已安装 (`pip install PyQt5`)
- [ ] 项目依赖已安装 (`pip install -r requirements.txt`)
- [ ] 开发者已完成新 UI 开发

### 启动应用
```bash
cd E:\openclaw-projects\multi-platform-publisher
python ui/main_window_v2.py
```
