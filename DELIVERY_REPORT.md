## 🚀 使用方式

### 启动应用

```bash
cd E:\openclaw-projects\multi-platform-publisher
python app.py
```

或双击 `run.bat`

### 配置 Cookie

1. 打开应用
2. 进入「Cookie 管理」标签页
3. 点击平台卡片的「编辑」按钮
4. 粘贴 Cookie 并保存

### 发布内容

1. 在「发布内容」标签页编写内容
2. 选择要发布的平台
3. 点击「🚀 一键发布」
