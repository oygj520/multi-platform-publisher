# ⚡ 快速开始指南 - 多平台发布助手

> 5 分钟上手，快速开始多平台内容发布

**预计时间**: 5-10 分钟  
**适合人群**: 新手用户  
**版本**: v1.0

---

## 📋 开始前准备

请确保您已准备好：

- ✅ Windows 10/11 电脑
- ✅ Python 3.9+（如未安装，约需 5 分钟）
- ✅ 稳定的网络连接
- ✅ 至少一个平台的账号（知乎/小红书/快手/抖音）

---

## 🚀 5 分钟快速上手

### 第 1 分钟：安装依赖

**步骤 1**: 打开项目目录
```bash
cd E:\openclaw-projects\multi-platform-publisher
```

**步骤 2**: 运行安装脚本
```bash
setup.bat
```

**等待完成**，看到以下提示表示安装成功：
```
[1/3] 安装 Python 依赖... ✓
[2/3] 安装 Playwright 浏览器... ✓
[3/3] 完成！

现在可以运行 run.bat 启动应用
```

> 💡 **提示**: 首次安装可能需要 2-3 分钟，请耐心等待。

---

### 第 3 分钟：编写内容

**步骤 1**: 输入标题

在顶部的标题输入框中输入：
```
我的第一篇多平台发布文章
```

**步骤 2**: 编写正文

在 Markdown 编辑器中输入以下内容（支持 Markdown 格式）：

```markdown
# 欢迎来到我的空间

这是我用**多平台发布助手**创建的第一篇文章。

## 为什么选择这个工具？

- ✅ 一键发布到多个平台
- ✅ 支持 Markdown 格式
- ✅ 界面简洁美观
- ✅ 节省大量时间

## 使用体验

整体使用感受非常好，特别是：

1. **操作简单** - 只需几步即可完成发布
2. **效率高** - 不用重复登录各个平台
3. **格式统一** - 一次编辑，多平台同步

> 强烈推荐给需要多平台运营的朋友！

---
*发布于 2026 年*
```

**步骤 3**: （可选）选择封面图

1. 点击「📷 封面图片」旁边的「选择图片」按钮
2. 选择一张合适的图片
3. 图片路径会显示在按钮旁边

> 💡 **提示**: 封面图是可选的，但建议添加以提高吸引力。

### 第 5 分钟：发布内容

**步骤 1**: 点击发布按钮

点击右侧的「🚀 一键发布」按钮

**步骤 2**: 等待发布完成

您会看到：
- 进度条开始走动
- 状态文字变为「⏳ 正在发布到 zhihu...」
- 逐个平台显示发布进度

**步骤 3**: 查看发布结果

发布完成后会弹出提示框：
```
✅ 发布完成

成功发布到 2 个平台！

标题：我的第一篇多平台发布文章
平台：zhihu, xiaohongshu
```

同时，发布记录会自动添加到「📜 发布历史」标签页。

## ⚠️ 注意事项

### 首次使用必读

1. **当前版本状态**
   - ✅ 界面完整，功能可用
   - ✅ Markdown 编辑器正常工作
   - ✅ 发布记录功能正常
   - ⚠️ 实际发布功能需要配置 Cookie
   - ⚠️ 部分平台发布逻辑仍在开发中

2. **测试建议**
   - 首次使用建议先测试发布功能
   - 使用测试账号进行验证
   - 确认无误后再使用主账号

3. **Cookie 配置**
   - 如需实际发布，必须配置 Cookie
   - 详见 [USER_GUIDE.md](USER_GUIDE.md) 的 Cookie 配置教程
   - Cookie 仅存储在本地，不会上传

4. **发布限制**
   - 各平台有发布频率限制
   - 建议间隔 5-10 分钟发布一次
   - 不要短时间内大量发布

## 🎯 最佳实践

### 新手建议

1. **从简单开始**
   - 先发布短文章测试
   - 只选择 1-2 个平台
   - 确认流程后再发布重要内容

2. **内容准备**
   - 提前写好 Markdown 格式内容
   - 准备好合适的封面图
   - 检查标题是否吸引人

3. **时间选择**
   - 避免平台高峰期发布
   - 选择目标受众活跃时间
   - 工作日优于周末（视平台而定）

### 进阶技巧

1. **批量发布**
   - 准备好多篇内容
   - 依次发布到不同平台
   - 注意平台间隔时间

2. **内容适配**
   - 知乎：深度专业内容
   - 小红书：生活方式 + 精美图片
   - 快手/抖音：短视频 + 简短文案

3. **数据跟踪**
   - 定期查看发布历史
   - 记录各平台表现
   - 优化发布策略

## 🆘 遇到问题？

### 快速排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 应用无法启动 | Python 未安装 | 安装 Python 3.9+ |
| 缺少模块 | 依赖未安装 | 运行 `setup.bat` |
| 发布失败 | Cookie 失效 | 重新获取 Cookie |
| 界面无响应 | 程序繁忙 | 等待片刻，勿重复点击 |
