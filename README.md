# 🚀 多平台发布助手

一个桌面级的多平台内容发布工具，支持一键发布到知乎、小红书、快手、抖音。

## ✨ 功能特性

### MVP 版本 (v1.0)

- ✅ **Markdown 编辑器** - 支持实时预览
- ✅ **封面图上传** - 拖拽选择图片
- ✅ **多平台选择** - 知乎、小红书、快手、抖音
- ✅ **一键发布** - 自动发布到多个平台
- ✅ **发布记录** - 历史记录查看

### 后续版本 (v2.0+)

- 📝 定时发布
- 📝 格式自适应
- 📝 数据统计
- 📝 Cookie 管理
- 📝 AI 辅助

## 🛠️ 安装

### 方法一：直接运行（开发模式）

1. **安装 Python 3.9+**
   ```bash
   python --version
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **运行应用**
   ```bash
   python run.bat
   ```

### 方法二：使用 EXE（生产模式）

1. **打包应用**
   ```bash
   python build.py
   ```

2. **运行可执行文件**
   ```
   dist/MultiPlatformPublisher.exe
   ```

## 📖 使用说明

### 快速开始

1. **打开应用**
   - 运行 `run.bat` 或 `MultiPlatformPublisher.exe`

2. **编写内容**
   - 在编辑器中输入标题和内容（支持 Markdown）
   - 选择封面图片（可选）

3. **选择平台**
   - 勾选要发布的平台（知乎、小红书、快手、抖音）

4. **一键发布**
   - 点击「🚀 一键发布」按钮
   - 等待发布完成

### 平台配置

**注意**: 当前版本需要手动配置各平台 Cookie

1. 进入「⚙️ 设置」标签页
2. 填写对应平台的 Cookie
3. 保存配置

```
multi-platform-publisher/
```

## ⚠️ 注意事项

1. **Cookie 安全**: Cookie 存储在本地，请妥善保管
2. **发布限制**: 各平台有发布频率限制，请勿频繁发布
3. **内容规范**: 请遵守各平台内容规范
4. **测试建议**: 首次使用建议先测试发布
